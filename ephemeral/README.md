# Ephemeral

Tracked operating material that daily or periodic jobs mine into durable docs,
skills, policy, manuals, and reference indexes.

Common subdirectories:

- `worklog/` for session worklogs.
- `reviews/` for consensus or independent review artifacts.
- `projects/` for per-project scratch artifacts gathered after the user starts
  or names a project. Each active project should be listed in `AGENTS.md` with
  its path, name, and a short description.
- `research/` for raw source captures and notes that are not ready to become
  durable docs.

This directory is ignored by routine ripgrep through `.rgignore`. Search it
deliberately during documentation maintenance.
